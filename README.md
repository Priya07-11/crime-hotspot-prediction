# Crime Hotspot Prediction 
About This Project:

This project shows which areas have more crime using easy-to-understand heatmaps. It helps anyone, from local authorities to curious citizens, see patterns and hotspots in a clear way.

How It Works:

The project takes crime data and turns it into colorful maps. Areas with more crime show up more intensely, while safer areas are lighter. This makes it simple to spot trends and make decisions based on real information.

How to Use It:

Follow these simple steps to see the results:

Install the required packages by opening your terminal and typing:

pip install -r requirements.txt


Run the heatmap script by typing:

python main.py


Open the results in the output folder. You will see maps showing areas with higher and lower crime intensity.

What You Will See:

* Clear and colorful heatmaps

* Areas with high crime highlighted

* Easy visuals that anyone can understand

Why It Matters:

Understanding crime patterns is important. These maps can help communities, authorities and individuals take informed actions and stay safe.
